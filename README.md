## Project Overview

**MindSage AI** is designed to support mental well-being through AI-assisted conversations that emphasize **empathy, trust, and emotional safety**.

Rather than replacing professional care, the platform acts as a **supportive companion**, offering reflective responses that help users process emotions, manage stress, and build healthier emotional awareness.

MindSage AI is ideal for users seeking a reflective, low-pressure space for emotional awareness and stress management.

## Features

- **AI-Powered Chat Sessions**: Supportive conversations designed around empathy and reflection.
- **Session-Based Conversations**: Users can create, revisit, and continue multiple chat sessions.
- **Emotional Safety First**: Language and interactions are designed to reduce pressure and avoid harmful claims.
- **Secure Authentication**: JWT-based authentication for protected user access.
- **Modern UI/UX**: Calm, accessible interface built with Tailwind CSS and motion-based micro-interactions.
- **Scalable Backend Architecture**: REST APIs with clean separation of concerns.
- **Robust Error Handling**: Graceful fallbacks for API failures and network issues.

## Tech Stack

- **Frontend**:

  - Next.js (App Router)
  - TypeScript
  - Tailwind CSS
  - Framer Motion
  - Lucide Icons
  - React Markdown
  - Date-fns

- **Backend**:
  - Node.js
  - Express.js
  - TypeScript
  - MongoDB + Mongoose
  - JWT (Authentication)
  - Google Gemini API

## Usage

### As a User:

1. Sign up or log in securely to access the platform.
2. Start a new therapy session or continue an existing one.
3. Share thoughts, emotions, or concerns in a safe environment.
4. Receive empathetic, AI-guided responses in real time.
5. Explore guided wellness activities when stress signals are detected.
6. View and manage past chat sessions for ongoing reflection.
